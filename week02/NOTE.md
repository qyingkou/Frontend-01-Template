### 作业A：写一个正则表达式 匹配所有 Number 直接量
> 作业说明
```
打开NumberLiteral.html，看正则表达式公式
打开devtool-console面板，查看测试结果
```
> 分析如下：
```
参考：ECMA文档的《11.8.3 Numeric Literals》章节

Number字面量的定义：
- 二进制的组成
    - 0b 二进制数值
    - 0B 二进制数值
- 八进制的组成
    - 0o 八进制数值
    - 0O 八进制数值
- 十进制的组成
    - 十进制整数 . 十进制小数 十进制指数(可选)
    - . 十进制小数 十进制指数(可选)
    - 十进制整数 十进制指数(可选)
- 十六进制
    - 0x 十六进制数值
    - 0X 十六进制数值

/* 组成分析 */
// 二进制的组成部分
    二进制数值 ::= /[0-1]+/
// 八进制的组成部分
    八进制数值 ::= /[0-7]+/
// 十六进制的组成
    十六进制数值 ::= /[0-9a-fA-F]+/
// 十进制的组成部分
    十进制指数 ::= /[eE][1-9][\d]*/
    十进制整数 ::= /0|[1-9][\d]*/
    十进制小数 ::= /[\d]*/
```

- <PS> 。 
- \ <EscapeSequence> 。
- <LineContinuation> 。 /(?:\u000A)|(?！（\u000A)\u000D)|\u2028|\u2029|\u000D\u000A/u
    - <LF>
    - <CR>[lookahead ≠ <LF>]
    - <LS>
    - <PS>
    - <CR><LF>
/* 双引号内的字符串内容 */

/* 公共的字符串内容 */
EscapeSequence ::= 
    - /[^\'\"\\\b\f\n\r\t\v]|(?:\u000A|\u000D|\u2028\u2029)/
    - /[^\d]*0/
    - /x[0-9a-fA-F]{2}/
